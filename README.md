# KeshavAi

KeshavAi is a Visual Studio Code extension that provides custom right-click commands for project scaffolding and automation.

## Features

- Right-click commands in the Explorer
- Create folders or files quickly
- Modular command structure

## Requirements

- Visual Studio Code >= 1.109.0
- No separate Node.js installation required

## Usage

Right-click in the Explorer:

- Workspace root → runs root command
- Folder → runs folder command

## Requirements

- Visual Studio Code version **1.109.0 or newer**
- No separate Node.js installation required (VS Code includes its own Node runtime)
